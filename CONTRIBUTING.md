# Cómo Contribuir a OpenClinicIA

## Proceso
1. Fork del repositorio
2. `git checkout -b feat/descripcion-corta`
3. Seguí los estándares en `brain_OC/02_ESTANDARES_CODING.md`
4. `pytest` / `npm run test` — cobertura mínima 80%
5. Commit semántico: `feat(agenda): descripcion`
6. PR contra `main`

## Áreas donde se necesita ayuda
- 🏥 Médicos/clínicos — revisión de prompts SOAP
- 🌎 Traductores — i18n EN, PT
- 🧪 QA — casos de prueba clínicos
- 🔧 DevOps — Helm charts, CI/CD
